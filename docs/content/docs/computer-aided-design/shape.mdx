# Boolean Operations on Shapes
You can perform boolean operations such as union and cut on shapes. These operations modify the shape in-place and return self for method chaining.

```python
# Single shape union
box1.union(box2)

# Multiple shapes union
box1.union([box2, box3, box4])

# Cut (subtraction)
box1.cut(hole)

# Method chaining
result = box1.union(box2).cut(hole)
```

# Exporting
Shapes can be exported to various file formats for use in other applications:

```python
# Export to STL (mesh format)
shape.to_stl("output.stl")

# Export to STEP (CAD format)
shape.to_step("output.step")

# Export to PNG (rendered image)
shape.to_png("output.png", view="iso")
```

# Visualization
Render shapes to PNG images with different camera views:

```python
# Isometric view (default)
shape.to_png("iso.png", view="iso")

# Orthographic views
shape.to_png("front.png", view="front")
shape.to_png("top.png", view="top")
shape.to_png("right.png", view="right")

# Custom size
shape.to_png("large.png", width=1920, height=1080)
```
