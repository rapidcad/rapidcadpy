# Installation

Install RapidCADpy using pip:

```bash
git clone https://github.com/rapidcad/rapidcadpy.git
cd rapidcadpy
pip install -e .
```

# Basic Usage

Here's a simple example to create a basic 3D model using the new App approach:

```python
from rapidcadpy.integrations.occ import OpenCascadeApp

# Initialize OpenCascade application
app = OpenCascadeApp()
app.new_document()

# Create a workplane on the XY plane
wp = app.work_plane("XY")

# Create a simple cube
cube = wp.move_to(-5, -5).rect(10, 10).extrude(10)

# Export to STEP format
cube.export("my_cube.step")
```

## More Advanced Example

Here's how you can create and cut a feature using the fluent API and the Inventor backend:

```python
app = InventorApp()
app.new_document()
wp = app.work_plane("XY")

arm_thick = 5

# Create an arm shape
arm = (
    wp.move_to(-4.5, 0)
    .line_to(-4.5, 20)
    .line_to(-8, 45)
    .three_point_arc((0, 53), (8, 45))
    .line_to(4.5, 20)
    .line_to(4.5, 0)
    .three_point_arc((0, -4.5), (-4.5, 0))
    .extrude(arm_thick)
)

# Create a hole and cut it from the arm
hole = wp.move_to(0, 45).circle(4).extrude(arm_thick)
arm.cut(hole)
```
