Set the current backend.

<PySourceCode >

```python
def __set__(self, obj, value):
    """Set the current backend."""
    if isinstance(value, str):
        success = _backend_manager.set_backend(value)
        if not success:
            available = list(_backend_manager.get_available_backends().keys())
            raise ValueError(
                f"Backend '{value}' is not available. Available backends: {available}"
            )
    else:
        raise TypeError("Backend must be a string")
```
