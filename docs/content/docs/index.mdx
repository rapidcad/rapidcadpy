# RapidCAD.py Documentation

Welcome to RapidCAD.py - a powerful Python library for creating parametric 3D CAD models and performing structural analysis.

## What is RapidCAD.py?

RapidCAD.py provides a fluent, intuitive API for:

- **Parametric CAD Modeling** - Create complex 3D models with simple Python code
- **FEA Analysis** - Perform structural analysis with torch-fem integration
- **Multiple Backends** - Support for Autodesk Inventor and Open CASCADE
- **Export Formats** - Export to STEP, STL, and other standard formats

## Quick Example

```python
from rapidcadpy.integrations.inventor import InventorApp

# Initialize application
app = InventorApp()
app.new_document()

# Create a simple beam
wp = app.work_plane("XY")
beam = wp.move_to(0, 0).rect(10, 20).extrude(100)

# Export the model
beam.export("beam.step")
```
