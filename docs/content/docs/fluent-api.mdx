# Fluent API Examples

RapidCADpy provides a fluent API that allows you to create CAD models using an intuitive, chainable interface. This guide demonstrates various examples of creating 3D models using the fluent API.

## Inventor Backend and App Approach

RapidCADpy now supports a backend-specific `App` class for direct integration with Inventor. This enables you to manage documents and workplanes in a way that closely matches Inventor's workflow.

### Setting up an Inventor App and Workplane

```python
from rapidcadpy.integrations.inventor import InventorApp

# Initialize Inventor application
app = InventorApp()
app.new_document()

# Create a workplane on the XY plane
wp = app.work_plane("XY")
```

## Creating and Modifying Shapes

### Creating an Arm and Cutting a Hole

```python
app = InventorApp()
app.new_document()
wp = app.work_plane("XY")

arm_thick = 5

# Create an arm shape
arm = (
    wp.move_to(-4.5, 0)
    .line_to(-4.5, 20)
    .line_to(-8, 45)
    .three_point_arc((0, 53), (8, 45))
    .line_to(4.5, 20)
    .line_to(4.5, 0)
    .three_point_arc((0, -4.5), (-4.5, 0))
    .extrude(arm_thick)
)

# Create a hole and cut it from the arm
hole = wp.move_to(0, 45).circle(4).extrude(arm_thick)
arm.cut(hole)
```

## Simple Shapes

### Creating a Cube

```python
app = InventorApp()
app.new_document()
wp = app.work_plane("XY")
cube = wp.move_to(-5, -5).rect(10, 10).extrude(10)
cube.export("cube.step")
```

### Creating a Cylinder

```python
app = InventorApp()
app.new_document()
wp = app.work_plane("XY")
cylinder = wp.move_to(0, 0).circle(5).extrude(20)
```

## Best Practices

- Use `InventorApp()` to manage Inventor documents and workplanes.
- Chain operations fluently for readable and concise code.
- Use named planes like `"XY"`, `"XZ"`, or `"YZ"` for standard workplanes.
