# GMSH
When using GMSH for meshing, ensure that the `GMSH_EXECUTABLE_PATH` environment variable is set to the correct path of the GMSH executable on your system. For example, if you have GMSH installed via Miniconda, you can set it as follows:
```bash
export GMSH_EXECUTABLE_PATH=/opt/homebrew/Caskroom/miniconda/base/envs/ocp/bin/gmsh
```
